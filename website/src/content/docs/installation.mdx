# 安装

import { Tabs } from 'nextra/components'

## 系统要求

- Go 1.21 或更高版本（仅源码安装时需要）
- 任意支持 LSP stdio 协议的编辑器

---

## 安装方式

<Tabs items={['go install（推荐）', '源码编译', '预编译二进制']}>
  <Tabs.Tab>

使用 `go install` 直接安装到 `$GOPATH/bin`：

```bash
go install github.com/zerx-lab/LspProxy@latest
```

安装后二进制名称为 `LspProxy`。建议创建软链接或别名以便使用：

```bash
# 创建软链接（需要 $GOPATH/bin 在 PATH 中）
ln -s $(go env GOPATH)/bin/LspProxy $(go env GOPATH)/bin/lsp-proxy
```

  </Tabs.Tab>
  <Tabs.Tab>

克隆仓库后手动编译：

```bash
git clone https://github.com/zerx-lab/LspProxy.git
cd LspProxy
go build -o lsp-proxy .
```

将编译产物移动到 PATH 下的任意目录：

```bash
sudo mv lsp-proxy /usr/local/bin/
# 或放到用户目录
mv lsp-proxy ~/.local/bin/
```

  </Tabs.Tab>
  <Tabs.Tab>

从 [GitHub Releases](https://github.com/zerx-lab/LspProxy/releases) 下载对应平台的预编译二进制，解压后放入 PATH：

```bash
# Linux (amd64) 示例
curl -L https://github.com/zerx-lab/LspProxy/releases/latest/download/lsp-proxy-linux-amd64.tar.gz | tar xz
sudo mv lsp-proxy /usr/local/bin/
```

  </Tabs.Tab>
</Tabs>

## 验证安装

```bash
lsp-proxy --help
```

输出应类似：

```
LspProxy 透明代理 LSP 消息，将文档注释实时翻译为中文。

使用方式：
  # 直接运行代理（将 LSP 命令作为参数传入）
  lsp-proxy -- rust-analyzer

  # 传入额外参数给 LSP 服务器
  lsp-proxy -- clangd --background-index

  # 打开 TUI 管理界面
  lsp-proxy --tui

  # 指定翻译引擎
  lsp-proxy -e openai -- rust-analyzer
```
